# My Claude Web App

A playground for various demo apps, built with React + Vite (TypeScript).

## Live URLs

- **Vercel**: https://my-claude-web-app.vercel.app
- **Cloudflare Pages**: https://my-claude-web-app.pages.dev

## Development

```bash
npm install
npm run dev
```

## Tech Stack

- React + TypeScript
- Vite
- Vercel (auto-deploy on push to `main`)
- Cloudflare Pages (auto-deploy on push to `main`)
